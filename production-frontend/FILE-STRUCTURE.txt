## Key Design Patterns Used

1. **Standalone Components** - All components are standalone (no NgModule)
2. **OnPush Change Detection** - Improved performance
3. **Reactive Forms** - Type-safe form handling
4. **Signals** - Angular's new state management
5. **Dependency Injection** - Using inject() function
6. **Service Pattern** - Centralized API communication
7. **Smart/Presentational Split** - Separation of concerns
8. **Dialog Pattern** - Forms in modals
9. **Observable Pattern** - Async data handling
